Budgie is a desktop environment that uses GNOME technologies such as GTK+ (> 3.x) and is developed by the Solus project as well as by contributors from numerous communities such as Arch Linux, Manjaro and Ubuntu Budgie. Budgie's design emphasizes simplicity, minimalism and elegance.

Applications[edit]

See also: List of GTK+ applications

Budgie desktop tightly integrates with the GNOME stack, employing underlying technologies to offer an alternative desktop experience. Budgie applications generally use GTK and header bars similar to GNOME applications. Budgie builds what is effectively a Favorites list automatically as the user works, moving categories and applications toward the top of menus when they are used.[1]

History[edit]

Budgie was initially developed as the default desktop environment for the Evolve OS Linux distribution. After the name change of Evolve OS to Solus,[2] development of Budgie accelerated.

Early versions of Budgie were slow and prone to crashes.[3] Speed and reliability have improved with time.[4]

Budgie v1 was released on February 18, 2014 and v10 on December 27, 2015. The versioning scheme since changed, with the current release being 10.5.

Adoption[edit]

Arch
Budgie Desktop is available in Arch Linux's Community repository[5] as well as distributed as budgie-desktop-git in the Arch User Repository.
Debian
Budgie Desktop is available in Debian's main repositories in the Stable branch since Debian 9.
GeckoLinux
GeckoLinux, an openSUSE-based distribution, provides GeckoLinux Budgie.[6]
Manjaro
In November 2015, the Manjaro Community announced the availability of Manjaro Budgie 15.11,[7] on June 14, 2016, the availability of Manjaro Budgie 16.06.1.[8]
Solus
Solus's primary desktop environment is Budgie.[1][9]
SparkyLinux
Budgie Desktop is available at the current repository of SparkyLinux
Ubuntu Budgie
The first version of the Ubuntu Budgie distribution was 17.04; it is an official Ubuntu flavor. Ubuntu Budgie 17.04 differs from the original Ubuntu 17.04 distribution only in that Budgie is the default desktop interface.[10] As of June 2017, Ubuntu Budgie was among the most popular Ubuntu variants.[11]
Void
Budgie Desktop is available at the current repository of Void Linux (January 2018).[12]
