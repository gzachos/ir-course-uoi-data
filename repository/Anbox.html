Anbox is a free and open-source compatibility layer that aims to allow mobile applications and mobile games developed for Android to run on GNU/Linux distributions.[2] Canonical introduced Anbox Cloud, for running Android applications in a cloud environment.[3]

Technical overview[edit]

It executes the Android runtime environment by using LXC (Linux Containers), recreating the directory structure of Android as a mountable loop image, whilst using the native Linux kernel to execute applications. It makes use of Linux namespaces trough LXC for isolation. Applications do not have any direct hardware access, all accesses are sent through the Anbox daemon.[4]
