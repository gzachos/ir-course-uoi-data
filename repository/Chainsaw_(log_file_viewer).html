Chainsaw is a java-based GUI software tool to view and analyze computer log files - specifically logs generated by the Log4j logging system. Both Log4j and Chainsaw are Open source projects under Apache Software Foundation. The latest release is Chainsaw v2.[2] Chainsaw can read log files formatted in Log4j's XMLLayout, receive events from remote locations, read events from a DB, or work with JDK 1.4 logging events.
