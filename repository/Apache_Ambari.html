Apache Ambari is a software project of the Apache Software Foundation.[2] Ambari enables system administrators to provision, manage and monitor a Hadoop cluster, and also to integrate Hadoop with the existing enterprise infrastructure. Ambari was a sub-project of Hadoop but is now a top-level project in its own right.

Ambari is used by companies including IBM, Hortonworks, Cardinal Health, EBay, Expedia, Kayak.com, Lending club, Neustar, Macy's, Pandora Radio, Samsung, Shutterfly, and Spotify.[citation needed]
